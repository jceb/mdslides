Make your first Slideshow

Get your first slideshow up and running in less than 5 minutes.

  1. Create an empty repository at GitHub
  2. Add a new file called SLIDES.md to the repository
  3. Paste the following content into the file:
    # My first markdown slideshow

Author: Your Name

## Agenda

1. Markdown is easy to read
2. Markdown is easy to write
3. Let's use it for presentations

## The End

Thank you for your time.

  5. Copy the link to your presentation and share it with friends
